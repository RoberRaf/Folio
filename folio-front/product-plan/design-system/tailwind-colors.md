# Tailwind Color Configuration

## Color Choices

- **Primary:** `rose` — Used for buttons, links, key accents, CTAs
- **Secondary:** `pink` — Used for tags, highlights, secondary elements, decorative accents
- **Neutral:** `stone` — Used for backgrounds, text, borders, cards

## Usage Examples

Primary button: `bg-rose-600 hover:bg-rose-700 text-white`
Secondary badge: `bg-pink-100 text-pink-800 dark:bg-pink-950 dark:text-pink-300`
Neutral text: `text-stone-600 dark:text-stone-400`
Neutral background: `bg-stone-50 dark:bg-stone-950`
Neutral border: `border-stone-200 dark:border-stone-800`
